# AI Driven Monitoring System

---

## Problem Domain
- I live in and manage a public living space where people are free to come and go. People will often leave trash lying around, forcing residents to clean up after them.

---

## Solution Domain
- Set up a monitoring system that keeps track of who comes and goes through the living space and keep track of what they are bringing and leaving in the living space.

---

## Technology Stack
- Raspberry Pi (Hosts Server Programs)
- Personal Computer (Laptop, or ANY PC capable of running Java)

---

## Software Structure
- Python script running a OpenCV based computer vision model (Raspberry Pi)
- Java program that serves video clip data to clients (Raspberry Pi)
- Client Java program that recevies the server data (Client PC)

---

## Sprint 2
- Full camera and computer vision implementation
- Data transfer, sending video files to client program

---

## Rough Schedule
